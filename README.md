# Monk's Shops

## Installation
Simply use the install module screen within the FoundryVTT setup

## Usage
This module is to provide some pregenerated shops for DnD5e that work with Monks Enhanced Journals.
Aswell as adding some roll tables for common items.  So you can create your own shop quickly.
